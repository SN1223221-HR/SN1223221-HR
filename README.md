# Hi there, I'm an HRBP who codes. 👋

**「エンジニアリング組織の採用・組織開発」を専門とする HRBP / 採用担当**

従来の人事業務（オペレーション）をテクノロジー（GAS, API, SQL）で自動化し、**「人間同士の対話」や「組織文化の醸成」といった本質的な業務に時間を最大化すること**をミッションとしています。

---

## 🚀 Value Proposition

私の強みは、**HR（人事戦略）**と**Engineering（技術実装）** の交差点に立ち、自ら手を動かして課題を解決できる点です。

* **Operational Excellence**: 採用フローのボトルネックを特定し、API連携やスクリプトで物理的に工数を削減します。
* **Data-Driven Decision**: SQLを用いたファネル分析により、感覚値ではない「数字に基づいた採用戦略」を立案します。
* **Empathy for Engineers**: エンジニアの文化や開発プロセスを理解しているため、現場と乖離のない採用要件定義やオンボーディング設計が可能です。
